## Project Overview (Loyiha haqida)

Ushbu loyihada **UCI Machine Learning Repository**dan olingan "Myocardial Infarction Complications" ma'lumotlar to'plami tahlil qilindi. 
Loyiha davomida quyidagi muhim bosqichlar amalga oshirildi:

1. **Data Preprocessing:** 111 ta tibbiy ko'rsatkichdagi bo'sh kataklar (NaN) median usulida to'ldirildi va ma'lumotlar standartlashtirildi.
2. **Dimensionality Reduction:** PCA (Principal Component Analysis) yordamida 111 ta ustun 2 ta asosiy komponentaga (PC1 va PC2) qisqartirildi.
3. **Feature Loadings:** PC1 ga eng kuchli ta'sir ko'rsatuvchi omillar (EKG ritmi va Qon bosimi) aniqlandi.
4. **Visualization:** Bemorlarning ahvoli (barqaror va xavfli) 2D xaritada (Radar) ranglar orqali aks ettirildi.
